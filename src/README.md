# Ghost Note

为任意窗口（App）绑定便签（Note）

## 如何使用

- `ghost`, `notes`, `gn` 关键字浏览便签列表
    - 点击 App 名称可打开对应应用
- `Bind Ghost Note` 中任意关键字或`Tab`键激活便签创建，新创建的标签绑定于`uTools`激活前的活动窗口（App）
    - `ctrl-w` 返回列表（便签将自动保存）
